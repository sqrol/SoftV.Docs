# Monitoring

The **Monitoring** page is used to inspect the robot's state. Here you can manage state variables that you've added in your program.
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_2.png"/>
</div>

On the right you will see a list of variables that you can drag onto the main panel to work with them:
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_2_drag.png"/>
</div>

There are buttons at the bottom for working with the panel elements:
- Clear all elements from the panel
- Import panel elements
- Export panel elements
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_2_low_buttons.png"/>
</div>

Controls for sorting and filtering variables in the list are available here:
<div style={{textAlign: 'left'}}>
<img src="/docshome/img/shufflecad/shufflecad_2_sort.png"/>
</div>
